# Bookstore

React School project.

Technologies:

- language: **Typescript**
- Class Components
- Redux-Saga
- Fake Api to simulate database
- platform: **web**

> Running the project

First terminal

```
$ cd client
$ npm start
```

Second terminal

```
$ cd server
$ json-server books.db.json --port 3001 --watch
```
